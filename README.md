#  Calculator-C#

A simple calculator application built with **C# Windows Forms** that supports the four basic arithmetic operations with a clean and user-friendly interface.

##  Features

-  Addition
-  Subtraction
-  Multiplication
-  Division
-  Dark Mode
-  Light Mode
-  Input Validation
-  Modern and Simple UI

##  Validation

The application validates user input by:

- Preventing empty values.
- Allowing only numeric input.
- Preventing division by zero.
- Displaying clear error messages for invalid operations.

##  Technologies Used

- C#
- .NET Framework
- Windows Forms (WinForms)
